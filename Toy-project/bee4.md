# bee4 (beeFour)

## bee4 의미
- 꿀벌과 Net의 조합
- 꿀벌은 영어로 bee이고, Network의 Net은 한글발음이 넷인데, 숫자 4도 넷이라고 발음함

- Network는 상호협력을 최우선으로 여김
- 즉, bee4은 우두머리에 의한 협력이 강조되는 네트워크 시스템을 의미함
