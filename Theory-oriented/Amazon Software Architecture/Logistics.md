# AGV를 연계한 GTP
- [영상보기](https://www.youtube.com/watch?v=flGG8F-2OGk)
- 출고 지시를 받은 GV가 랙을 자동 운반
- 랙이 스테이션에 도착하면 PPS가 피킹 대상 제품의 위치와 수량 표시
  - 작업자는 손쉽게 제품 토탈 피킹
- 효율적 공간 구성, 피킹과 분류를 짧은 동선 내에서 수행
- 다품종 소량 물류 특성은 가진 e커머스, 의류, 제약 등 적용 가능
- 피킹 후 분류 작업을 수행하여 작업자 동선 최소화
